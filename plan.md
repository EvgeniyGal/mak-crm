Build the CRM system using the following stack and architecture choices:

- **Frontend**: Fast development and modern tooling.
- **UI Components**: Tailwind CSS for styling, complemented by a modern UI component library for efficient form handling and data fetching.
- **Backend**: Supabase as the backend service, providing a managed PostgreSQL database, authentication, and real-time capabilities.
- **Authentication**: Utilize Supabase Auth for user authentication, supporting email/password login and Google OAuth.
- **Responsive Design**: Ensure UI layouts adapt for mobile and desktop, with optimal readability and usability across device sizes.
- **UI/UX Quality**: Use modern design patterns — lightweight modals, intuitive filtering panels, data tables with inline badge indicators (e.g., "2/8" for subscription usage), minimal clutter, and clear visual hierarchy.
- **Modal Windows**: Integrate modern and attractive modal components to enhance user interaction and visual appeal.
- **Deployment**: Host the frontend on a platform like Vercel and utilize Supabase's managed services for the backend.
- **Localisation** The system must support multilingual UI (English and Ukrainian), with Ukrainian set as the default language and a language switcher available.
